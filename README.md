# Sales Data Analysis Project

## Project Overview

This project analyzes Superstore sales data to identify business trends, customer behavior, and sales performance.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

## Dataset

The project uses the Superstore Sales Dataset containing 9,800 sales records and 18 columns.

## Analysis Performed

1. Calculated Total, average, highest, and lowest sales.
2. Analyzed Sales by category.
3. Analyzed Sales by region.
4. Identified Top 10 products by sales.
5. Visualized category-wise sales analysisusing bar chart.
6. Visualized Monthly sales trend analysis.

## Key Findings

* Technology generated the highest sales revenue.
* The West region achieved the highest sales.
* Several products contributed significantly to overall revenue.
* Sales showed fluctuations with periods of strong growth.
